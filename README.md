# BirthdayOS v12.0 

A minimal, interactive birthday website built for a young developer.

Inspired by the editorial aesthetic of Dice.fm, BirthdayOS combines bold typography, smooth animations, and an interactive developer terminal to celebrate Version 12.0.

## Features

- Minimal black & white design
- Foggy-inspired typography
- Smooth Framer Motion animations
- Interactive developer terminal
- Hidden secret commands
- Dark mode easter egg
- Fully responsive layout
- Built with React + Tailwind CSS + Vite

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion

## Getting Started

Clone the repository:

```bash
git clone <repository-url>
cd birthday-v2
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```txt
http://localhost:5173
```

## Build for Production

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
